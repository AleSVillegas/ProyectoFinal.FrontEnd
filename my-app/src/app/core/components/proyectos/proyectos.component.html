Título del Proyecto 2

Incluyo este mismo proyecto por haber sido un desafío y en el cual que apliqué mis conocimientos en diferentes tecnologías para crear mi propio Portfolio Web.
A través de esta experiencia, pude desarrollar habilidades en el campo de la programación y el desarrollo web.
Para comenzar, se diseño el prototipo de la página. Luego, implementé al Front End tecnologías como HTML, CSS y Bootstrap. Estas tecnologías me permitieron crear una maqueta estática de mi Portfolio Web, aplicando principios de diseño web y personalizando el aspecto visual de la página.
Continué desarrollando el Front End dinámico utilizando el framework Angular. Utilicé TypeScript para crear una Single Page Application (SPA) y aprendí a utilizar componentes, servicios y proveedores para diseñar mi página de manera interactiva.
En cuanto al Back End, utilicé el lenguaje Java y el framework Spring Boot. Esto me permitió desarrollar las funcionalidades de mi Portfolio Web y crear APIs REST para comunicarme con la base de datos.
También implementé una arquitectura en capas y apliqué conceptos de seguridad y autenticación.
En el ámbito de las bases de datos, utilicé MySQL para diseñar y crear una base de datos relacional. Identifiqué los datos relevantes para mi proyecto y establecí relaciones entre las entidades de la base de datos.
Durante el proceso de desarrollo, también utilicé herramientas de control de versiones como Git y GitHub para administrar y realizar seguimiento de los cambios en mi código. Esto me permitió trabajar de manera colaborativa y tener un historial de versiones de mi proyecto.
Finalmente, en la etapa de Deploy, apliqué conocimientos en DevOps. Realicé pruebas para asegurar la calidad de mi aplicación, implementé medidas de seguridad web y realicé la comunicación efectiva de mi proyecto. En resumen, mi proyecto de Portfolio Web me permitió aplicar una variedad de tecnologías, como HTML, CSS, Bootstrap, TypeScript, Angular, Java, Spring Boot, MySQL, UML y herramientas como Figma, Git y GitHub, entre otras.
A través de esta experiencia, desarrollé habilidades en diseño web, programación Front End y Back End, bases de datos, Programación Orientada a Objetos y DevOps.
