Proyecto Final de Robótica

Este proyecto integra la comunicación Modbus entre un "BRAZO MÉCANICO UR5" y un "MIR100" utilizando un PLC LOGOSOFT 8 conectado en la misma red.

El proceso se inicia con un pulsador que cambia un registro en el MIR100, lo cual activa su recorrido, de transporte de figuras geométricas, hacia la estación del brazo robot UR5. Una vez que el MIR100 llega a la estación del robot, cambia un registro indicando que ha alcanzado la posición deseada.

El programa del BRAZO MÉCANICO UR5 espera una entrada de registro enviada por el MIR100 al llegar a una posición preseleccionada. Al recibir este registro, el UR5 se posiciona en un punto de paso previamente definido y envía una salida de registro hacia el MIR100 para que se detenga en dicha posición.

El UR5 se posiciona en Home y captura imágenes del área de trabajo utilizando una cámara 3D. Mediante algoritmos de procesamiento de imágenes, se detectan y clasifican figuras geométricas.

Una vez detectada la figura, el UR5 se desplaza hacia las coordenadas correspondientes, sujeta la figura con su Gripper y la coloca en la posición adecuada.

Posteriormente, se envía un registro al MIR100 para reanudar la misión. El MIR100 regresa a su estación de carga y, al llegar, cambia un registro para que el PLC detecte el fin del proceso y envíe la señal de STOP.

El UR5 se mueve hacia la posición de depósito de la figura y luego regresa a su posición de inicio.

Portfolio Argentina Programa

Decidí incluir este proyecto por considerarlo un desafío personal, transmitiendo los conocimientos que fui adquiriendo a lo largo de mi trayecto en el curso y abarcando diversas tecnologías en el desarrollo de mi propio Portfolio Web.

Comencé por diseñar el prototipo de la página y luego implementé tecnologías de Front-End como HTML, CSS y Bootstrap para crear una maqueta estática y atractiva visualmente.

Continué el desarrollo del Front-End utilizando el framework Angular y aprovechando TypeScript para crear una experiencia interactiva y amable para los usuarios.

En cuanto al Back-End, utilicé Java y el framework Spring Boot para desarrollar las funcionalidades del sitio web y crear APIs REST para interactuar con la base de datos. También implementé medidas de seguridad y autenticación para garantizar la protección de la información.

En el ámbito de las bases de datos, utilicé MySQL para diseñar y crear una estructura de base de datos, definiendo relaciones entre las entidades y garantizando una gestión efectiva de los datos.

Durante todo el proceso de desarrollo, utilicé herramientas de control de versiones como Git y GitHub para gestionar los cambios y mantener un historial de versiones del proyecto.

Finalmente, en la etapa de despliegue (Deploy), apliqué prácticas de DevOps para garantizar la calidad de la aplicación, realizar pruebas para asegurar una comunicación óptima.

En síntesis, este proyecto fue una oportunidad para aplicar una amplia gama de tecnologías, desde HTML, CSS y Bootstrap hasta TypeScript, Angular, Java, Spring Boot y MySQL. Además, adquirí experiencia en diseño web, programación Front-End y Back-End, gestión de bases de datos y DevOps.
